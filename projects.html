COptimism

Python, PyGame, Tkinter GUI, Minimax Algorithm

COptimism is a chess engine programmed in python, with a user interface built with PyGame. It implements the AI using the minimax algorithm, which takes in a position on the board and evaluates it based on material advantage, piece activity, centre control, king safety, etc. Then, the algorithm is passed to the AI, who views each possible position at a depth of the user's choice and makes its move.

Motion Sensor Lamp

Python, Raspberry Pi, Circuits

This automatic motion sensor lamp was built with a Raspberry Pi, along with PIR (passive infrared) sensors, relay switches, an old lamp after soldering, and backend programmed in python. When the sensors detect heat waves in front of it, it turns on the lamp. The time interval for shut-off and sensor sensitivity can be adjusted accordingly.
